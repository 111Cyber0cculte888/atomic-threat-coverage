### Sigma rule

```

status: experimental

    - https://car.mitre.org/wiki/CAR-2013-05-002

    - attack.defense_evasion

detection:
    selection:
        CommandLine:
            - "*:\\RECYCLER\\*"
            - "*:\\SystemVolumeInformation\\*"
            - "%windir%\\Tasks\\*"
            - "%systemroot%\\debug\\*"
    condition: selection

    - False positives depend on scripts and administrative tools used in the monitored environment

    product: windows

    selection:
        EventID: 1
```
